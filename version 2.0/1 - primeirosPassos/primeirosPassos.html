1. Tags:

1.1. A tag "i"

A tag i no HTML é usada para indicar que o texto contido dentro dela é um texto com ênfase ou destaque em relação ao texto em torno dele. Anteriormente, essa tag era comumente usada para itálico, mas com a evolução dos padrões web, seu uso para itálico foi substituído pela tag em.

Portanto, atualmente, a tag i é mais frequentemente usada para fins semânticos, como destacar um termo técnico, uma palavra em um idioma estrangeiro ou uma frase de um texto, sem implicar que o texto deve ser exibido em itálico. O estilo visual, incluindo a itálico, deve ser aplicado usando CSS.

Aqui está um exemplo simples para ilustrar a diferença entre i e em:

Isso é um texto com ênfase.

Isso é um texto com ênfase.

Neste exemplo, ambos os i e em podem ser exibidos como itálico por padrão no navegador, mas semanticamente eles têm significados diferentes. O i é mais genérico e pode ser usado para texto com destaque visual, enquanto o em é usado para ênfase semântica, indicando que a palavra é enfatizada no contexto da frase.

1.2. A tag strong e a tag b

A tag b é usada para estilizar texto de forma que ele apareça em negrito, mas não adiciona significado semântico. A tag strong, por outro lado, é usada para dar ênfase forte ao texto, indicando que o conteúdo é de maior importância.

3. Tipografia:

Tipografia no HTML refere-se ao estilo e aparência do texto em uma página web. Envolve a seleção de fontes, tamanhos, pesos, espaçamento e outras propriedades que afetam a legibilidade e a estética do texto. A tipografia é uma parte crucial do design web, pois influencia como o conteúdo é percebido e lido pelos usuários.

4. Listas Ordenadas e Não Ordenadas:

4.1. Lista Ordenada

Uma lista ordenada possui numeração para cada item da lista. Utilizando a tag "ol" faço uma lista ordenada, para adicionar os itens nessa lista eu coloco a tag "li" dentro da "ol". Conforme o exemplo abaixo:

  1. Item 1
  2. Item 2
  3. Item 3

É possível colocar uma lista ordenada dentro de uma "li". Conforme abaixo:

  1. item 1
  2. item 2
  3. item 3
  4. item 4
    1. item 4.1
    2. item 4.2
    3. item 4.3

4.2. Lista Não Ordenada

Uma lista não ordenada não possui numeração para cada item da lista e sim marcadores "bolinhas" ou outro tipo de marcador que desejar. Utilizando a tag "ul" faço uma lista não ordenada, para adicionar os itens nessa lista eu coloco a tag "li" dentro da "ul". Conforme o exemplo abaixo:

  • Item A
  • Item B
  • Item C

5. Links:

5.1. Links Internos e Externos

O links é pra colocar uma conexão com alguma outra pagina dentro da sua pagina inicial. Você pode incluir um link utilizando a tag a em seguida "colando" o endereço da pagina que voce deseja referenciar dentro do href, também é preciso dar um nome de indicação para o seu link, escrevendo entre os >/" do a. É possível configurar para abrir o link em uma nova página, escrevendo dentro do a a tag target e selecionando _blank, já utilizando o _self, ele fica no comportamento padrão, vai abrir na mesma pagina.

Exemplo:

Meu github (abrindo em nova página)

Meu github (abrindo na mesma página)
