1. Tag img

A tag img é usada para incorporar imagens em uma página HTML. Ela é auto-fechada e não possui tag de fechamento. O formato jpg comprime a imagem, mas perde um pouco da resolução, enquanto o png mantém a resolução, mas é mais pesado e tem a vantagem de suportar transparência. O jpg é mais adequado para fotos, enquanto o png é melhor para logotipos.

O formato svg define a imagem por vetores, permitindo que a imagem não perca resolução, mas é limitado a imagens opacas e sólidas.

Explicação:

  • img: Usada para adicionar imagens.
  • src: Caminho da imagem.
  • alt: Texto alternativo para acessibilidade.
  • width e height: Definem as dimensões da imagem.
  • title: Texto exibido ao passar o mouse sobre a imagem.

Explicação:

  • audio: Usada para adicionar áudio às páginas web.
  • controls: Adiciona controles de reprodução.
  • autoplay: Toca o áudio automaticamente ao carregar a página.
  • loop: Repete o áudio continuamente.
  • muted: Inicia o áudio no modo mudo.
  • preload: Controla o pré-carregamento do áudio.

4. Tag track

A tag track é usada para adicionar faixas de texto a elementos de mídia, como vídeos e áudios.

Vídeo com Legendas

Vídeo com Capítulos

Explicação:

  • src: Caminho para o arquivo de texto da faixa.
  • kind: Tipo de faixa (subtitles, captions, descriptions, chapters, metadata).
  • srclang: Idioma da faixa.
  • label: Título da faixa.
  • default: Define a faixa como padrão.

5. Tag iframe

A tag iframe é usada para incorporar outra página HTML dentro de uma página HTML.

Explicação:

  • width e height: Define a largura e altura do iframe.
  • sandbox: Aplica restrições de segurança ao conteúdo.
  • allow: Especifica quais recursos são permitidos.
  • allowfullscreen: Permite que o iframe entre em modo de tela cheia.
