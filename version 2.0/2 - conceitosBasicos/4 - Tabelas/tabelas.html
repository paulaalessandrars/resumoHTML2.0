1. Tag table

A tag <table> em HTML é usada para criar tabelas que organizam dados em linhas e colunas. É uma das formas mais comuns de exibir informações tabulares na web. Aqui estão alguns conceitos básicos sobre como a tag <table> funciona:

  1. Estrutura Básica: A tag <table> define o início de uma tabela. Dentro dela, você utiliza outras tags para estruturar a tabela:
    • <tr> (table row): Define uma linha na tabela.
    • <th> (table header cell): Define um cabeçalho de coluna ou linha na tabela (opcional).
    • <td> (table data cell): Define uma célula de dados na tabela.
  2. Atributos Importantes:
    • border: Define a largura da borda da tabela (geralmente em pixels).
    • cellpadding: Define o espaçamento interno das células.
    • cellspacing: Define o espaçamento entre as células.
    • width e height: Define a largura e altura da tabela, respectivamente.
    • align: Alinha a tabela à direção escolhida.
    • summary: Adiciona uma descrição à tabela, útil para leitores de tela.
  3. Exemplo Básico:
    Exemplo de Tabela Básica
    Nome Idade País
    João 30 Brasil
    Maria 25 Portugal
  4. Elementos Relacionados:
    • <thead>, <tbody>, <tfoot>: Elementos opcionais para organizar cabeçalhos, corpo e rodapé da tabela.
    • <caption>: Elemento opcional para adicionar um título descritivo à tabela.

A tag <table> é essencial para estruturar e apresentar dados tabulares de forma organizada e acessível em páginas web. Ela permite criar grids de informações onde cada célula pode conter textos, imagens, links ou outros elementos HTML.

2. Tag tr

A tag <tr> em HTML define uma linha dentro de uma tabela (<table>). Cada <tr> contém células de dados (<td>) ou células de cabeçalho (<th>). É usada para organizar e estruturar dados em colunas dentro de uma tabela.

Estrutura e Uso Básico:
  • <tr> é abreviação de "table row", ou seja, "linha da tabela".
  • Dentro de uma <tr>, você pode incluir células de dados (<td>) ou células de cabeçalho (<th>).
  • As células são organizadas automaticamente em colunas, alinhadas com as células correspondentes em outras linhas.
Exemplo:
Nome Idade País
João 30 Brasil
Maria 25 Portugal
Paula 30 Brasil
Julia 24 Brasil
Pedro 24 Espanha

Neste exemplo:

  • A primeira <tr> define uma linha de cabeçalho com três células de cabeçalho (<th>): "Nome", "Idade" e "País".
  • As <tr> seguintes definem linhas de dados com três células de dados (<td>) cada, representando informações sobre "João" e "Maria".
Atributos e Estilização:
  • <tr> pode ter atributos como class e id para estilização e seleção via CSS ou manipulação via JavaScript.
  • É comum usar CSS para estilizar <tr> e suas células (<td> e <th>), ajustando cores de fundo, bordas, margens, entre outros.

3. Tag td e th

  • <td> (table data cell): A tag <td> é usada para definir células de dados dentro de uma tabela. Cada <td> representa uma célula individual que contém dados.

    Exemplos:
    João 30 Brasil
    Maria 25 Portugal
    Atributos e Estilização:
    • colspan: Define o número de colunas que a célula deve abranger.
    • rowspan: Define o número de linhas que a célula deve abranger.
    • Estilização CSS: Você pode aplicar estilos às células <td> usando CSS para ajustar cores, bordas, espaçamento, entre outros.

  • <th> (table header cell): A tag <th> é usada para definir células de cabeçalho dentro de uma tabela. As células <th> geralmente contêm títulos ou rótulos que descrevem o conteúdo das colunas ou linhas.

    Exemplos:
    Nome Idade País
    Atributos e Estilização:
    • scope: Define o escopo da célula de cabeçalho, pode ser "col" (coluna), "row" (linha), "colgroup" (grupo de colunas) ou "rowgroup" (grupo de linhas).
    • Estilização CSS: As células <th> podem ser estilizadas para se destacar visualmente das células <td> usando CSS, como negrito, alinhamento centralizado, cores de fundo, entre outros.

4. Tag tbody, thead e tfoot

As tags <thead>, <tbody> e <tfoot> em HTML são usadas para estruturar tabelas de forma mais semântica e organizada. Elas ajudam a separar o conteúdo da tabela em diferentes seções, facilitando a leitura e a manutenção do código.

  • <thead> (table header): Define o cabeçalho da tabela. Geralmente contém células de cabeçalho (<th>) que descrevem as colunas.
  • <tbody> (table body): Define o corpo da tabela. Contém linhas de dados (<tr>) com células de dados (<td>).
  • <tfoot> (table footer): Define o rodapé da tabela. Pode conter células de resumo ou informações adicionais.
Exemplo Completo:
Nome Idade País
João 30 Brasil
Maria 25 Portugal

5. Estilizando Tabelas com CSS

Estilizar tabelas com CSS permite que você melhore a aparência visual e a usabilidade das suas tabelas. Aqui estão algumas dicas e técnicas comuns para estilizar tabelas usando CSS:

Estilização Básica:
Nome Idade País
João 30 Brasil
Maria 25 Portugal
Dicas Adicionais:
  • Use a propriedade border-collapse para colapsar as bordas da tabela, criando um visual mais limpo.
  • Adicione :hover em <tr> para destacar a linha que o cursor está passando por cima, melhorando a interatividade.
  • Utilize :nth-child(even) para alternar as cores das linhas, criando um efeito zebra que facilita a leitura.
